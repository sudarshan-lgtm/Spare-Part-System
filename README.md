# Spare Parts Management System

## 📌 Overview
A complete product-based ordering and inventory management system designed for businesses to manage spare parts efficiently.

## 🚀 Features
- Product listing with images
- Advanced search (Part number, Company, Category)
- Multi-role system (Admin, Customer, Member, Warehouse)
- Order routing & dispatch system
- Excel upload for bulk product management
- Ledger management (Receivable & Transactions)
- Role-based product visibility

## 🛠 Tech Stack
- PHP (Core)
- MySQL
- JavaScript, AJAX
- HTML, CSS

## 💼 Use Case
This system is actively used in real-world business operations to manage product orders and warehouse dispatch efficiently.
